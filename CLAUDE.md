# ARA Project Instructions

## Ruflo Agent Orchestration (MANDATORY)
- Always use Ruflo for multi-agent orchestration and swarm coordination.
- Ruflo is globally installed and registered as an MCP server — available in every session.
- For parallel agents, swarm tasks, or multi-step workflows, prefer `ruflo` commands and MCP tools.
- Key commands: `ruflo start`, `ruflo agent`, `ruflo swarm`, `ruflo task`, `ruflo mcp`.
